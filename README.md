# Fresh OS X install
This is a WIP script to setup all the software and my [dot files](https://github.com/codeinabox/dotfiles) on a fresh installation of OS X.

## Dependencies
You need the following things installed before hand
 * Homebrew
 * Ansible 2

## Usage
Simply run `make`
